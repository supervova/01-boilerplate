# 01-boilerplate

Class-light CSS-фреймворк на принципах разумной достаточности. Тут нет стилей на все случаи жизни — только первой необходимости.

Class-light CSS framework based on the principles of reasonable sufficiency. There are no styles for all occasions, only the essentials.

- Class-light MVP
- Design Tokens
- PostCSS + Behasis (block, element and `has`/`is` modifiers)
- Cascading Layers
- HTML `dialog`, `details`, `datalist`, `[type='date']`
- Snippets Collection
