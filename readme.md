# 01-boilerplate

Class-light CSS framework based on the principles of reasonable sufficiency. There are no styles for all occasions, only the essentials.

- Class-light MVP
- Design Tokens
- PostCSS
- Cascading Layers
- HTML `dialog`, `details`, `datalist`, `[type='date']`
- 'Behasis' Naming Convention: `block__element` + `has`/`is` modifiers
- Snippets Collection
